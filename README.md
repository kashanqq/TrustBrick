# TrustBrick 🚀

Это проект — простейший счетчик на блокчейне Solana с использованием фреймворка **Anchor** и фронтенда на **Next.js**.

---

## 🏗 Структура Проекта

- `programs/` — Исходный код смарт-контракта (Rust/Anchor).
- `app/` — Фронтенд приложение (Next.js + TypeScript).
- `tests/` — Интеграционные тесты для смарт-контракта.
- `migrations/` — Скрипты для миграции/развертывания.

---

## 🛠 Предварительные Требования

Для запуска вам понадобятся:
1. **Rust & Cargo** (1.75+)
2. **Solana CLI** (1.18.x)
3. **Anchor CLI** (0.29.0+)
4. **Node.js** (v18+) & **npm** / **yarn**

(https://solana.com/ru/docs/intro/installation)
---

## 🧱 Как запустить смарт-контракт

1. **Сборка программы:**
   ```bash
   anchor build
   ```

2. **Запуск тестов:**
   ```bash
   anchor test
   ```

3. **Локальный деплой (Localhost):**
   - Запустите локальный валидатор: `solana-test-validator` (в отдельном терминале).
   - Разверните программу: `anchor deploy`.

> [!TIP]
> Убедитесь, что ваш `Anchor.toml` указывает на нужную сеть (localnet / devnet).

---

## 🌐 Как запустить фронтенд

Перейдите в папку `app` и установите зависимости:

```bash
cd app
npm install
# или
yarn install
```

Запустите сервер разработки:

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
